    Bounded queue (maxsize=10) per RESEARCH.md Pitfall 2 — prevents unbounded growth when no clients are connected.

    **3. Replace `_log_reading` callback** with a new callback that:
    - Updates `state.last_speed_kmh`, `state.last_power_w`, `state.last_cadence_rpm` from the IndoorBikeData reading
    - Builds the snapshot dict with exactly these 6 keys:
      ```python
      snapshot = {
          "speed_kmh": reading.speed_kmh,
          "power_w": reading.power_watts,
          "cadence_rpm": reading.cadence_rpm,
          "gear": state.gear_engine.current_gear,
          "real_grade_pct": state.real_grade_percent,
          "effective_grade_pct": state.gear_engine.effective_grade(state.real_grade_percent),
      }
      ```
    - Calls `broadcast_queue.put_nowait(snapshot)` wrapped in try/except `asyncio.QueueFull` (drop oldest: `broadcast_queue.get_nowait()` then retry `put_nowait`)
    - Still calls `_log_reading(reading)` for terminal output

    Name the new callback `_on_reading` and define it inside `main()` so it captures `state` and `broadcast_queue` via closure:
    ```python
    def _on_reading(reading: IndoorBikeData) -> None:
        state.last_speed_kmh = reading.speed_kmh
        state.last_power_w = reading.power_watts
        state.last_cadence_rpm = reading.cadence_rpm
        snapshot = {
            "speed_kmh": reading.speed_kmh,
            "power_w": reading.power_watts,
            "cadence_rpm": reading.cadence_rpm,
            "gear": state.gear_engine.current_gear,
            "real_grade_pct": state.real_grade_percent,
            "effective_grade_pct": state.gear_engine.effective_grade(state.real_grade_percent),
        }
        try:
            broadcast_queue.put_nowait(snapshot)
        except asyncio.QueueFull:
            broadcast_queue.get_nowait()
            broadcast_queue.put_nowait(snapshot)
        _log_reading(reading)
    ```

    **4. Create ws_task** after gear_logger_task:
    ```python
    ws_task = asyncio.create_task(
        broadcast_loop(broadcast_queue, stop_event),
        name="ws_broadcast",
    )
    ```

    **5. Add ws_task to the shutdown gather:**
    Update the `asyncio.gather()` call to include `ws_task`:
    ```python
    await asyncio.gather(
        reconnect_task, consumer_task, gear_logger_task, ws_task,
        return_exceptions=True,
    )
    ```
    And update the cancel loop to include `ws_task`.

    **6. Pass `_on_reading` to telemetry_consumer:**
    Change `telemetry_consumer(queue, _log_reading)` to `telemetry_consumer(queue, _on_reading)`.

    **CRITICAL:** `_on_reading` is a plain `def` (not async). It uses only `put_nowait` / `get_nowait`. This preserves the BLE callback safety constraint.
